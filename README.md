# e-plantShopping

e-plantShopping is a web-based shopping cart application developed for **Paradise Nursery**, an online plant shop.

## Project Description
This application allows users to browse a variety of houseplants, view plant details including images, names, and prices, and add plants to a shopping cart. Users can manage cart items by increasing or decreasing quantities, removing items, and viewing the total cost dynamically.

## Features
- Paradise Nursery landing page with a welcome message
- Product listing page with categorized houseplants
- Shopping cart with add, remove, and update quantity functionality
- Dynamic cart item count
- Responsive navigation bar

## Technologies Used
- React.js
- Redux Toolkit
- JavaScript
- CSS
